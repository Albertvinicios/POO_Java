# ☕ POO em Java (Curso em Vídeo)

Repositório de estudos focado em Programação Orientada a Objetos (POO) em Java, baseado no curso do Curso em Vídeo, com implementações próprias, exemplos práticos e comentários explicativos.

---

## 🚀 Objetivo

Consolidar os conceitos de POO em Java através da prática, demonstrando:

* Entendimento de modelagem orientada a objetos
* Aplicação de conceitos fundamentais
* Organização e clareza no código

---

## 🧠 Conceitos abordados

* Classes e objetos
* Encapsulamento
* Herança
* Polimorfismo
* Abstração
* Métodos e atributos

---

## 🛠️ Tecnologias utilizadas

* Java

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/poo-java.git
```

2. Acesse a pasta do projeto:

```bash
cd poo-java
```

3. Compile o arquivo desejado:

```bash
javac NomeDoArquivo.java
```

4. Execute:

```bash
java NomeDoArquivo
```
